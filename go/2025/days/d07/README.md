# Advent of code challenge 2025, in GO, day d07

[Problem statement](https://adventofcode.com/2025/day/7)

For the exemple:
```
.......S.......
...............
.......^.......
...............
......^.^......
...............
.....^.^.^.....
...............
....^.^...^....
...............
```

In part1, we just trace beams:
```
.......S.......
.......|.......
......|^|......
......|.|......
.....|^|^|.....
.....|.|.|.....
....|^|^|^|....
....|.|.|.|....
...|^|^|||^|...
...|.|.|||.|...
```
And the result is the number of splits. .

But for part 2, we "charge" the beams with the number of timelines it appears in:

```
.......S.......
.......1.......
......1^1......
......1.1......
.....1^2^1.....
.....1.2.1.....
....1^3^3^1....
....1.3.3.1....
...1^4^634^1...
...1.4.634.1...
```
And the result is the sum of all timelines for each beam at the end.
